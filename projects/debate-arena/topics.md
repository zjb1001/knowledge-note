# 示例辩题库

## 科技类

1. **人工智能是否会取代人类工作**
   - 正方：AI将大幅提升效率，取代重复性工作
   - 反方：AI创造新岗位，人类有不可替代的创造力

2. **自动驾驶是否应该尽快普及**
   - 正方：可以减少交通事故，提高出行效率
   - 反方：技术不成熟，伦理法律问题未解决

3. **元宇宙是否是互联网的未来**
   - 正方：沉浸式体验是下一代互联网形态
   - 反方：概念炒作，实际应用有限

## 社会类

4. **996工作制是否合理**
   - 正方：奋斗精神，企业竞争力需要
   - 反方：违反劳动法，损害员工健康

5. **高考是否应该取消英语科目**
   - 正方：减轻学生负担，专注专业科目
   - 反方：全球化需要英语能力

6. **网红经济是否有利于社会发展**
   - 正方：创造就业，促进消费
   - 反方：价值观扭曲，泡沫经济

## 伦理类

7. **基因编辑技术是否应该用于人类**
   - 正方：可以消除遗传疾病
   - 反方：伦理风险， slippery slope

8. **动物实验是否应该被禁止**
   - 正方：动物也有权利，科技可以找到替代方案
   - 反方：医学进步需要，人类福祉优先

## 经济类

9. **数字货币是否会取代传统货币**
   - 正方：便捷高效，未来趋势
   - 反方：波动大，监管难，安全风险

10. **全球化是否利大于弊**
    - 正方：资源配置优化，经济效率提升
    - 反方：加剧不平等，文化同质化

---

## 使用示例

```bash
# 使用默认辩题
./start-debate.sh

# 使用自定义辩题
./start-debate.sh "元宇宙是否是互联网的未来"

# Python直接调用
python debate.py --topic "996工作制是否合理" --rounds 5
```
