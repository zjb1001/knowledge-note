## 模块分类

### 基础I/O类
- **PORT** - 引脚配置基础
- **DIO** - 数字量读写

### 模拟/数字转换
- **ADC** - 模数转换
- **PWM** - 数字量输出（模拟效果）

### 定时/测量类
- **GPT** - 通用定时
- **ICU** - 输入捕获
- **OCU** - 输出比较
- **WDG** - 系统监控

### 通信类
- **SPI** - 同步串行通信
- **CAN** - 车载网络通信
- **LIN** - 低成本串行通信
- **ETH** - 高速数据传输

### 存储类
- **FLS** - Flash存储

---

## 学习路径建议

1. **基础阶段**: PORT → DIO → ADC
2. **定时阶段**: GPT → PWM → ICU → OCU
3. **通信阶段**: SPI → CAN → LIN → ETH
4. **高级阶段**: WDG → FLS
