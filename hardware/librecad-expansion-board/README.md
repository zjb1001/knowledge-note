# LibreCAD 低成本扩展板设计方案

## 方案概述

使用开源 CAD 软件 LibreCAD 设计简单扩展电路板，为设备添加 **HDMI 接口** 和 **RJ-45 以太网接口**，导出 Gerber 文件后低成本打样制作。

| 评估维度 | 结论 | 说明 |
|---------|------|------|
| **技术可行性** | ✅ 可行 | LibreCAD 支持2D绘图，可导出 DXF/SVG，配合 KiCad 可完成 PCB 设计 |
| **成本控制** | ✅ 极低成本 | PCB 打样 5片约 ¥20-50，元件成本 ¥10-30 |
| **复杂度** | ⚠️ 中等 | 需要配合 KiCad 进行电路设计和 Gerber 导出 |
| **适用场景** | ✅ 原型验证 | 适合快速原型、DIY 项目、教学演示 |

## 实施步骤

### 步骤1: 需求分析与选型

**确定接口类型：**
- **HDMI**: 标准 Type-A 母座，19pin
- **RJ-45**: 标准 8P8C 接口，带 LED 指示灯
- **连接方式**: 2.54mm 排针或 FPC 软排线

**主控板兼容性检查：**
- 确认主控板是否有 HDMI 信号输出引脚
- 确认是否有以太网 PHY 芯片或 RMII 接口
- 检查电平匹配（3.3V / 5V）

### 步骤2: 电路设计 (KiCad)

**LibreCAD 定位：**
> LibreCAD 是 **2D 机械制图软件**，不直接支持电路设计。
> 实际 workflow: LibreCAD(结构图) → KiCad(电路/PCB) → Gerber

**推荐 Workflow：**
```
LibreCAD (绘制板框/定位孔/机械结构)
    ↓ 导出 DXF
KiCad PCBNew (导入板框，进行电路设计)
    ↓ 布局布线
KiCad (生成 Gerber)
    ↓ 上传
嘉立创/捷配 (下单打样)
```

### 步骤3: 电路原理图设计

**HDMI 部分电路：**
```
主控板 HDMI 信号 → TMDS 信号线 → HDMI 母座
注意事项：
- 差分对阻抗控制 100Ω
- 建议加入 ESD 保护器件
- 3.3V 供电
```

**RJ-45 部分电路（以 RMII 为例）：**
```
主控板 RMII 接口 → 以太网 PHY 芯片 (如 LAN8720) → RJ-45 + 网络变压器
或
主控板 USB → USB 转以太网芯片 (如 RTL8152) → RJ-45
```

### 步骤4: PCB 布局 (LibreCAD + KiCad)

**在 LibreCAD 中绘制：**
```
1. 绘制板框轮廓 (如 80mm x 50mm)
2. 标注 HDMI 接口位置
3. 标注 RJ-45 接口位置  
4. 绘制安装孔位置 (3mm 孔径)
5. 导出 DXF 文件
```

**在 KiCad 中完成：**
```
1. PCBNew → File → Import → DXF (导入板框)
2. 放置 HDMI 封装 (HDMI-A-19P)
3. 放置 RJ-45 封装 (带变压器一体化模块)
4. 差分对布线 (HDMI TMDS 信号)
5. 电源/地线布线
```

### 步骤5: 导出 Gerber 文件

**KiCad 导出设置：**
```
File → Plot → Gerber
├── 层选择:
│   ├── F.Cu (顶层铜)
│   ├── B.Cu (底层铜)
│   ├── F.SilkS (顶层丝印)
│   ├── B.SilkS (底层丝印)
│   ├── F.Mask (顶层阻焊)
│   ├── B.Mask (底层阻焊)
│   └── Edge.Cuts (板框)
├── 格式: RS-274X
└── 输出目录: gerber/
```

**生成钻孔文件：**
```
File → Fabrication Outputs → Drill Files (.drl)
```

### 步骤6: 下单打样

**推荐平台：**
| 平台 | 价格 | 周期 | 优势 |
|------|------|------|------|
| 嘉立创 | ¥5/5片 | 3-5天 | 价格便宜，品质稳定 |
| 捷配 | ¥20/5片 | 5-7天 | 支持更多特殊工艺 |
| 华秋 | ¥30/5片 | 3-5天 | 在线 DFM 检查 |

**下单参数设置：**
```
板厚: 1.6mm
层数: 2层
颜色: 绿色 (免费)
铜厚: 1oz
最小线宽: 6mil (0.15mm)
最小孔径: 0.3mm
表面处理: 有铅喷锡 (便宜) / 无铅喷锡
```

## 风险提示 ⚠️

| 风险点 | 等级 | 应对措施 |
|--------|------|----------|
| HDMI 高速信号完整性 | 🔴 高 | 必须控制差分阻抗，建议4层板 |

| 焊接难度 | 🟡 中 | HDMI 接口间距小，需一定焊接技巧 |
| 信号兼容性 | 🟡 中 | 确认主控板电平和协议支持 |

## 优化建议

### 方案A: 极简版 (推荐新手)
- 仅实现 HDMI 或 RJ-45 其中一个
- 使用转接板方案，不设计复杂电路
- 单层板设计，成本最低

### 方案B: 标准版 (推荐)
- HDMI + RJ-45 同时实现
- 2层板设计，嘉立创免费打样
- 加入 ESD 保护和电源滤波

### 方案C: 增强版
- 4层板设计，保证信号完整性
- 加入 HDMI 电平转换芯片
- 加入电源管理模块

## 结论

**方案可行，但需要注意：**

1. **LibreCAD 仅适合做机械结构图**，电路设计需配合 KiCad
2. **HDMI 高速信号**需要阻抗控制，建议使用4层板或短走线
3. **成本控制优秀**，整体可在 ¥30 以内完成
4. **适合场景**：原型验证、DIY 项目、教学演示

**推荐实施路径：**
```
LibreCAD 画板框 → KiCad 设计电路 → 嘉立创打样 → 手工焊接测试
```
